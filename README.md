# Deep Mapping Dresden - NLP Analysis of Travellogues about Dresden from 1586 to 1899

## Description
A Study of Traveller's Ideas of Dresden from 1560-1900 based on the database provided by the Institut für Sächsische Geschichte und Volkskunde (isgv) https://reise.isgv.de/.

## Installation
Simply install the requirements with pip install requirements.

## Usage
Run `python server.py` and open `http://127.0.0.1:5000`in your browser.

## File Structure
In raw metadata you will find files which I complied more or less by hand from the isgv Reiseberichte site queried for Dresden: https://reise.isgv.de/places/Q1731.
