Deep Mapping Dresden - Travelogues 1604-1900
Map

About This Project

This interactive map visualizes place mentions in historical travel writings about Dresden. The map allows exploration of how locations were described by travelers over time. The map was created as part of the study program "Digital Humanities M.A." by Luise Pohlmann in March 2026.

Data

The travelogues used for this visualisation come from the digital collection of travelogues concerning Saxony provided by the Institut für Sächsische Geschichte und Volkskunde (isgv).
The Metadata was scraped from their site and the images of the travelogue pages were transribed using a large language model. Next, place names were extracted using a different large language model. Finally, the place names were georeferenced by looking up their names either using either Open Street Map, or in cases where this did not yield results, using nominatim. Some places could not be georeferenced, in these cases, a large language model was used to try to correct transcription errors or modernize the spelling. These are the places marked as "uncertain" on the map. Overall 1025 place mentions from 67 travelogues written between 1604 and 1900 are included (see fulll ist in the bibliography below).
For further information on the data and scripts, see the repository for this project on github or send me an email Luise Pohlmann.

How to Use

  • Use the year slider to explore references chronologically.
  • Click map features to read quotes mentioning that place.
  • The sidebar shows what people wrote about "Dresden" in general.
  • The points fade in color as their mentions fade into the past.
